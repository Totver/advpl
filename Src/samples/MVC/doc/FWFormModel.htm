Classe FWFormModel

Descrição:
Classe principal de controle das regras de negócio ( model ) dos  formulários de edição. Cria um modelo estruturado que pode ser baseado em formulário de edição por campo ou grid. Esta classe permite o encapsulamento das regras de negócio b
Autor:
Eduardo Riera
Release:
P10 R1.4
Data:
22/10/2008
Observação:
Entende-se por formulário de edição toda interface, visual ou
não, que captura ou exibi os dados ao usuário de um sistema. Na versão
atual permite a composição de 'n' formulários de edição de campo e/ou grid's.
Um formulário de edição por campos permite a operação em apenas
uma linha da tabela (Master), enquando que formulário de edição
por grid permite a operação em diversas linhas da tabela (Detail)

Descrição:
Atribui uma mensagem de erro ao modelo. Quando uma mensagem de erro é atribuida, ela somente pode ser retirada pelo  evento que a provocou. O controle de mensagens de erro do modelo, permite identificar o Id´s do formulário e campo de origem do
Parâmetros:

cIdForm :
Tipo: CHARACTER - [Optional]
Id do formulário em validação

cIdField :
Tipo: CHARACTER - [Optional]
Id do campo do formulário em validação

cIdFormErr :
Tipo: CHARACTER - [Optional]
Id do formulário em que ocorreu o erro

cIdFieldErr :
Tipo: CHARACTER - [Optional]
Id do campo do formulário em que ocorreu o erro

cId :
Tipo: CHARACTER - [Optional]
Id da mensagem de help

cMessage :
Tipo: CHARACTER - [Optional]
Mensagem de erro

cSoluction :
Tipo: CHARACTER - [Optional]
Mensagem de solução

xValue :
Tipo: VARIANT - [Optional]
Valor especificado do campo

xOldValue :
Tipo: VARIANT - [Optional]
Valor anterior do campo

FWFormModel:SetOperation

Descrição:
Indica ao modelo a operacao de manipulacao de dados que será realizada. A classe utiliza esta informação para definir como serão inicializadas as informações do modelo. 
Parâmetros:

nOperation :
Tipo: NUMERIC
Indica a operação que será realizada com o modelo, sendo:
[1] View
[3] Insert
[4] Update
[5] Delete
[6] only Update
