FWFormModelStruct:AddField

Descrição:
Adiciona um campo a estrutura de metadado da classe 
Parâmetros:

cTitulo :
Tipo: CHARACTER
Titulo do campo

cTooltip :
Tipo: CHARACTER
Tooltip do campo

cIdField :
Tipo: CHARACTER
Id do Field

cTipo :
Tipo: CHARACTER
Tipo do campo

nTamanho :
Tipo: NUMERIC
Tamanho do campo

nDecimal :
Tipo: NUMERIC - [Optional - Default: 0]
Decimal do campo

bValid :
Tipo: BLOCK - [Optional - Default: {|| .T.}]
Code-block de validação do campo

bWhen :
Tipo: BLOCK - [Optional]
Code-block de validação when do campo

aValues :
Tipo: ARRAY - [Optional - Default: {}]
Lista de valores permitido do campo

lObrigat :
Tipo: LOGICAL - [Optional - Default: .F.]
Indica se o campo tem preenchimento obrigatório

bInit :
Tipo: BLOCK - [Optional]
Code-block de inicializacao do campo

lKey :
Tipo: LOGICAL - [Optional - Default: .F.]
Indica se trata-se de um campo chave

lNoUpd :
Tipo: LOGICAL - [Optional - Default: .F.]
Indica se o campo pode receber valor em uma operação de update.

lVirtual :
Tipo: LOGICAL - [Optional - Default: .F.]
Indica se o campo é virtual
