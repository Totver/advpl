Programa: FWTableDDL.prx
Classe FWTableDDL

Descrição:
Fornece metodos de manipulação da estrutura de dados de uma tabela.  Esta classe deve ser utilizada somente pelos programas de Framework e  tem como objetivo simplificar o relacionamento com as outras equipes  de framework da TOTVS 
Autor:
Eduardo Riera/Rodrigo Godinho
Release:
1.1
Data:
14/01/2009



Índice de Métodos

FWTableDDL:Activate
   Habilita o uso da classe, indicando que esta configurada e pronto para uso 

FWTableDDL:AddDbAcess
   Adiciona a configuração de conexão com o DbAcess. 

FWTableDDL:AlterTable
   Altera a tabela conforme definicao da nova estrutura 

FWTableDDL:ClassName
   Fornece o nome da classe 

FWTableDDL:CreateTable
   Cria a tabela conforme definicao de sua estrutura 

FWTableDDL:DeActivate
   Desabilita o uso da classe 

FWTableDDL:DropTable
   Elimina todos os registros de uma tabela. 

FWTableDDL:New
   Método construtor da classe 

FWTableDDL:OpenTable
   Realiza a abertura de uma tabela em modo ISAM 

FWTableDDL:PurgeTable
   Elimina todos os registros de uma tabela. 

FWTableDDL:SetTableStruct
   Atribui o objeto com a estrutura da tabela
