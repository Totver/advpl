Programa: FWTableStruct.PRX
Classe FWTableStruct

Descrição:
Fornece um objeto com a estrutura de uma tabela. Esta classe deve ser utilizada somente pelos programas de Framework e tem como objetivo simplificar o relacionamento com as outras equipes de framework da TOTVS 
Autor:
Eduardo Riera
Release:
1.1
Data:
14/01/2009
