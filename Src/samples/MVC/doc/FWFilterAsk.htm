Programa: FWFilterAsk.PRW
Classe FWFilterAsk

Descrição:
Fornece um objeto que permite montar filtros através de perguntas ao usuário 
Autor:
Eduardo Souza
Release:
10.1.1.4
Data:
08/04/2009



Índice de Métodos

FWFilterAsk:Activate
   Método de ativação da classe 

FWFilterAsk:AddAsk
   Adiciona uma pergunta para o filtro 

FWFilterAsk:ClassName
   Retorna o nome da classe (FWFILTERASK) 

FWFilterAsk:DeActivate
   Método destrutor da classe 

FWFilterAsk:GetAnswer
   Retorna a resposta de uma determinada pergunta 

FWFilterAsk:GetExpression
   Retorna a expressão do filtro 

FWFilterAsk:GetProfileID
   Retorna o identificador do filtro com pergunta utilizada na gravação  das configurações no profile do usuário. Deve ser utilizado quando  existir mais de um filtro com pergunta para a rotina. 

FWFilterAsk:LoadProfile
   Carrega as respostas do usuário salvas no profile 

FWFilterAsk:New
   Método construtor da classe 

FWFilterAsk:SaveProfile
   Salva as respostas do usuário no profile 

FWFilterAsk:SetAnswer
   Atualiza a resposta de uma determinada pergunta 

FWFilterAsk:SetExpression
   Indica a expressão do filtro que será substituida pelas respostas do  usuário. Para que os valores sejam localizados e substituidos, os  identificadores deverão estar entre percentes ("%").  

FWFilterAsk:SetIDProfile
   Define o identificador do filtro utilizado na gravação das configurações no profile do usuário. Deve ser utilizado quando existir mais de um filtro na rotina. 

FWFilterAsk:SetNoView
   Indica que não será exibida a tela de perguntas para o usuário 

FWFilterAsk:SetProfile
   Indica se a resposta deverá ser gravada no profile do usuário 

FWFilterAsk:UpdateExpression
   Atualiza a expressão do filtro de acordo com as respostas do usuário.

Eduardo Souza

10.1.1.4
Data:
