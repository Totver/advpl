Classe FWTableDML

Descrição:
Fornece metodos de manipulação dos dados de uma tabela, tais como  insert, update e deleted. Esta classe deve ser utilizada somente pelos programas de Framework e  tem como objetivo simplificar o relacionamento com as outras equipes  d
