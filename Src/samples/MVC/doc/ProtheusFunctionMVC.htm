Function FWFormCommit

Descrição:
Esta função realiza os tratamentos necessários a gravação dos formulários de edição do Microsiga Protheus. A gravacao  é realizada em niveis onde o primeiro elemento do modelo e posteriormente seus filhos são gravados. O  controle de transação é
Parâmetros:

oModel :
Tipo: OBJECT
Objeto do tipo FWFormModel

bBefore :
Tipo: BLOCK - [Optional - Default: {|| .T.}]
Bloco de código que será executado antes da gravação de cada linha das tabelas que compoem o formulário
de edição. O bloco de código recebe como parametro o objeto do model, o ID do formulário que esta sendo
gravado e o Alias.
Exemplo: {|oModel,cID,cAlias| teste(oModel,cID,cAlias)}

bAfter :
Tipo: BLOCK - [Optional - Default: {|| .T.}]
Bloco de código que será executado apos da gravação de cada linha das tabelas que compoem o formulário
de edição. O bloco de código recebe como parametro o objeto do model, o ID do formulário que esta sendo
gravado e o Alias.
Exemplo: {|oModel,cID,cAlias| teste(oModel,cID,cAlias)}

bAfterSTTS :
Tipo: BLOCK - [Optional - Default: {|| .T.}]
Bloco de código que será executado ao final da transacao. O bloco de código recebe como parametro
o objeto do model.
Exemplo: {|oModel| teste(oModel)}
@result lRetorno Indica se a transação foi bem realizada corretamente
Autor:
Eduardo Riera
