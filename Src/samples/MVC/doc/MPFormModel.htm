Classe MPFormModel

Descrição:
Fornece um objeto com o modelo hierárquico dos campos de edição para o Microsiga Prothues.  Esta classe é uma herança da classe modelo FWFormModel, utilizada apenas para facilitar o desenvolvimento do padrão MVC 
Autor:
Eduardo Riera
Release:
P10 R1.4
Data:
22/10/2008

   Método de desativação da classe. Quando a classe é desativada, os dados do formulário de edição são perdidos e os metódos de parametrização são ligados. Este método deve ser utilizado para reparametrizar a classe ou para reinicializar
